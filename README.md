# Brain Games

This project is a set of simple console games written in Java.
Each game asks the user questions and checks their answers.
To win, the player needs to give three correct answers in a row.

Games:
- Even - determine whether a number is even
- Calculator - solve random arithmetic expressions
- GCD - find the greatest common divisor
- Progression - find the missing number in an arithmetic progression
- Prime - determine whether a number is prime

---


##  How to run

### Requirements
- Java 17+
- Make
- Git

### Run the game

```bash
make run-dist
```
